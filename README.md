# C++ Console Projects

This repository contains two simple console-based C++ projects:

---

## 🏦 1. Bank System

A basic banking system that allows a user to:

### Features

* Create an account (username + PIN)
* Deposit money
* Withdraw money
* Display account information (secured with PIN)

---

## 🧠 2. Programming Quiz System

A simple quiz application to test basic programming knowledge.

### Features

* Multiple-choice questions (MCQ)
* Input validation (only accepts a, b, c, d)
* Tracks correct and wrong answers
* Displays results after each attempt
* Allows user to retry the quiz
* Counts number of attempts

---

## ⚙️ How to Run

### Compile:

```bash
g++ main.cpp -o app
```

### Run:

```bash
./app
```

---

## 📌 Notes

* Both projects are written using basic C++ (no OOP).
* Suitable for beginners learning:

  * Conditions
  * Loops
  * Arrays
  * Vectors
  * Input validation
